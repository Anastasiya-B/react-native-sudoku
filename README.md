## Features

1. Animations
2. Editing Mode
3. Game Resume

## Resources

1. sudoku generator: https://github.com/dachev/sudoku, just replace `underscore` with `lodash`
2. all the icons are downloaded from http://www.iconfont.cn/

## Build

1. clone this repo
2. `npm install`
3. `npm run pod`
4. `npm run ios` or `npm run android`
